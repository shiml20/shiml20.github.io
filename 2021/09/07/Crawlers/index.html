BS4

The full name of BS4 is Beautiful Soup. It provides easy and python type methods to process navigation, search, change parse tree and so on. It is a tool box. It can parse documents and change the input document into Unicode coding automatically, the output document into utf-8 encoding.

In other word Beautiful Soup is a library that get data from HTMLfiles.

object in BS4
  1. Tag
  2. BeautifulSoup
  3. NavigableString
  4. Comment
  5. element.ResultSet you can take it as a dictionary that is covered by list. Each element in it is a tag.
